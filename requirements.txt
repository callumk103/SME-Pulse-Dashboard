# SME Pulse — Business Health Dashboard

**A FunTech Project · Atlantic Technological University**

SME Pulse is a business health dashboard that helps SME owners monitor financial performance, spot risks early, and make better decisions — without needing financial expertise.

## Features

- Business Health Score, Cash Runway, KPI Cards with sparklines
- Deviation Alerts, Revenue Trends, Cost Breakdown, Cash Flow Timeline
- Owner/Board View Toggle for different audiences
- Google Sheets backend for persistent data
- Excel template upload for easy data entry

## Quick Start

```bash
pip install -r requirements.txt
streamlit run app.py
```

Runs in demo mode with sample data. See the full Google Sheets setup guide in the project documentation.
